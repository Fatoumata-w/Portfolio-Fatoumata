Pendu Explorer

Projet de groupe (4 personnes) réalisé dans le cadre du module Développement Web / JavaScript — Octobre 2025.

Objectif du projet

Créer un jeu du Pendu responsive avec deux niveaux de difficulté : Soft : mots faciles (cuisine, vie courante), Hard : mots longs et techniques (orthographe et accents). Le jeu devait être accessible sur tous les appareils et offrir une expérience utilisateur fluide.

Technologies utilisées

  • HTML5 / CSS3 / JavaScript (Vanilla)
  • Flexbox & CSS Grid pour la mise en page responsive
  • JSON pour les listes de mots (Soft / Hard)

Mon rôle

Participation à la partie GitHub, gestion des branches et du dépôt, ainsi que contribution au code JavaScript et à la logique du jeu.

Détails du projet

  • Durée : ~2 semaines
  • Période : Octobre 2025
  • Travail : En groupe de 4
  • Flow UX : Accueil → Choix du niveau → Jeu → Écran Victoire / Défaite
